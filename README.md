# unix_profiles_hmr
My version of unix profiles.

## How to use
```
# git clone https://github.com/hmr/unix_profiles_hmr.git
# cd unix_profiles_hmr
# ./install.sh
```

## After installation
### Install vim plugins
Start vim and type as below:
```
:PlugInstall
```
vimPlug will install plugins auto-magically.
