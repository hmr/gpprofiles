# unix_profiles_hmr
My version of unix profiles.

## How to use
```
git clone https://github.com/hmr/unix_profiles_hmr.git && \
ln -s unix_profiles_hmr/.vim* . && \
ln -s unix_profiles_hmr/.bashrc_history . && \
source .bashrc_history && \
echo source .bashrc_history >> .bashrc && \
tail .bashrc
```
